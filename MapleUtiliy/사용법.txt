[사용법]

1. Excutable 폴더 - MapleUtiliy.exe 실행
(★Excutable 폴더에 있는 platforms 및 dll은 프로그램 실행에 필요한 것들입니다. 삭제하면 안됩니다.★)

2. 메소 비율 가격 입력 및 셋팅 버튼 클릭

3. 추가 버튼 누르고, 아이템 이름, 캐시가격, 메소가격 입력 및 등록 => "추천▼" 버튼으로 캐시템 목록 추가

4. 등록이 되면 추가 버튼 아래 테이블에 한개씩 추가 됨

5. 정렬 버튼을 누르면 손익(메소) 순으로 바뀜

6. 아이템 이름, 캐시가격, 메소가격을 편집하면 손익(메소), 손익(원)이 자동으로 바뀜

7. 캐시 충전 할인율, 수수료 적용가능

8. 등록된 아이템 저장/불러오기(좌측상단 버튼) 기능 추가 => Resource/CustomDatas 폴더에 저장

9. ScrLk(스크롤락키)으로 스크린샷 찍기 기능 추가 => Screenshot 폴더에 저장

10. 수수료 3%로 했을경우 손익(메소) 수정 안되는 버그 수정
